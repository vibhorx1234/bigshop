# BigShop - Electronics Business Website

A full-stack marketing website for LG electronics business built with MERN stack (MongoDB, Express.js, React.js, Node.js) and Material-UI.

## Features

- 🏠 Home page with featured products
- 📱 Product categories (TV, Washing Machine, Refrigerator, Dishwashers, Microwave Ovens)
- 🔍 Advanced search, filter, and sort functionality
- 🖼️ Product modal with image gallery and video support
- 📧 Email and mobile verification (OTP)
- 📝 Enquiry forms with auto-email to owner
- 📦 Bulk enquiry system
- 🌓 Dark/Light theme toggle
- 🎨 Red and white color theme
- 📍 Google Maps integration for shop location
- ✨ Minimalist animations and vector backgrounds

## Tech Stack

### Frontend
- React.js
- Material-UI (MUI)
- React Router
- Axios
- Context API

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- Nodemailer (Email)
- Twilio (SMS/OTP)
- Express Validator
